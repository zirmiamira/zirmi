## Description du projet

Le projet Dressage est une simulation d’apprentissage d’animaux domestiques (chien et chat) développée en Java dans le cadre de l’UE Génie Logiciel et Projet (GLP).

Ce dépôt contient également un site web de présentation du projet, décrivant les objectifs, les spécifications, l’architecture et l’équipe.
